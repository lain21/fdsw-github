Mis trabajos

Desafío 2 - Construyendo un Landing Page

Diseñé una landing page con HTML y CSS externo, que incluye una imagen principal con texto, una sección de tres productos con descripciones, y un footer. Apliqué buenas prácticas de maquetación y organización del código.

Desafío 3 - Iguana Page

Desarrollé una landing page basada en una maqueta, utilizando HTML semántico y Flexbox para organizar el contenido. Implementé secciones con texto, íconos de Font Awesome, una galería de imágenes y cuidé la estructura del proyecto con archivos organizados en carpetas.

Desafío 4 - Cuppon

En este desafío desarrollé una página web responsiva usando Bootstrap. Apliqué el sistema de grillas para adaptar la vista a móviles y escritorio, utilicé componentes como card y formularios, e incorporé tipografías, colores e íconos según la maqueta.
